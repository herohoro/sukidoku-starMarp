- おはしを使う練習をするために手元を見せて話した
	1. まず、ペンの持ち方を確認してみる
	2. おはしと同じ動きになる縦の線を引く練習をしてみる
	3. ペンを少し人差し指に付けるように立てたまま、２本目のペンを入れてみる
	4. ２本目のペンは動かない。１本目のペンは縦の線を引く時と同じ動きをしていることが確認できる
	5. つかみやすい消しゴムをはさんでみるトレーニングをしてみる
